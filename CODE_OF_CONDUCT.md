# Code of Conduct

## Our pledge

We are committed to a respectful, harassment-free community for everyone.

## Our standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on technical merit and user impact

Examples of unacceptable behavior:

- Harassment, threats, or personal attacks
- Discriminatory or derogatory language
- Publishing private information without consent

## Enforcement responsibilities

Maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope

This code of conduct applies in repository issues, pull requests, discussions, and project communication channels.

## Reporting

Report conduct issues to `support@ignyte.solutions`. Reports are reviewed confidentially.
